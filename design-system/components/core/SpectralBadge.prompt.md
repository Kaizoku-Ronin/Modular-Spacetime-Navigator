A star's spectral classification as a colored pill. The color is derived from the leading class letter (O/B blue → G yellow → M red, D = white dwarf).

```jsx
<SpectralBadge type="G2V" />
<SpectralBadge type="M5.5V" />
<SpectralBadge type="DA" />
```

Pass the full type string; only the first letter drives the color. Use for catalog tables and star info panels.
